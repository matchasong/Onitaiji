# 🍑 ONITAIJI — 攻略ガイド

---

## はじめに

このゲームに「完全な正解」はありません。鬼殺し型と仲間思い型のどちらを使っても一長一短があり、状況に応じた判断が求められます。このガイドは「なぜそうするのか」の理由を含めて書いています。

---

## 基本的な数字感覚

まずこれを頭に入れてください。

```
初期財宝：8,000両
固定維持費：500両/ターン + 隊長数×100両/ターン
鬼殺し型のアイドルコスト：300両/ターン
仲間思い型のアイドルコスト：200両/ターン

初期状態（鬼殺し型+仲間思い型 2人）の最低コスト：
  500 + 200 + 300 + 200 = 1,200両/ターン

討伐依頼なしで放置すると：
  第1〜6ターンで 7,200両消費 → ほぼ破産
```

**アイドルコストを1ターンも出してはいけない**のはこのためです。

## 離脱の仕組みを理解する

離脱確率は士気と腕前（忠誠心の逆数）の掛け算です。

```
士気別の基礎確率：
  20未満  → 55%/ターン（ほぼ確実に去る）
  20〜34  → 30%/ターン
  35〜49  → 15%/ターン
  50〜64  → 5%/ターン
  65以上  → 1%/ターン

忠誠心による乗数（低いほど去りやすい）：
  忠誠心20以下（腕前80以上） → ×1.4
  忠誠心40以下（腕前60以上） → ×1.2
  忠誠心60以上（腕前40未満） → ×0.8

相性（chemistry）による補正（隠しパラメータ）：
  70以上 → ×0.8（去りにくい）
  25以下 → ×1.2（去りやすい）

英雄優遇中：×0.5（半減）
```

**最も危険な組み合わせ**は士気が低くて腕が立つ仲間です。
鬼殺し型の配下では腕前が毎ターン上がるため、士気が削れるほど優秀な者から去っていきます。

## 戦力スコアの設計

```
戦力スコア = 人数 × 平均スキル × 士気補正
               ↑           ↑           ↑
            量           質        コンディション

士気補正：
  75以上 → ×1.2
  50〜74 → ×1.0
  30〜49 → ×0.7
  30未満 → ×0.4

英雄ボーナス：スキル85以上が1人いるごとに戦力スコア+20%
```

### 必要スコアの目安

```
小型討伐  → 150〜200
中型討伐  → 350〜450
大型討伐  → 700〜900
```

士気が50を切ると戦力スコアが急落します。「人数もスキルもあるのに討伐が失敗する」のは士気補正が原因であることが多い。

## フェーズ別攻略

### 序盤（第1〜5ターン）— 生存

**最優先事項：アイドルコストを出さない**

第1ターンで鬼殺し型と仲間思い型を1討伐ずつアサインします。討伐の規模は以下を基準に選んでください。

```
鬼殺し型 → 小型か中型（大型は失敗リスク30%で序盤には重い）
仲間思い型 → 中型（一行のスキルが育つまで無理しない）
```

**第5ターンまでに鬼殺し型で大型討伐を1本完遂する**のが最重要課題です。第6ターンから長老評議会の圧力が発動するため、これを間に合わせないと2ターンごとに500両が自動で吸われ始めます。

序盤にやってはいけないこと：

- 失敗中に隊長交代する（失敗リスク×1.25、即座に悪化）
- 他村から助っ人を招く（維持費が増えるだけ、育成の時間もない）
- 戦力が大幅不足した状態で大型討伐を受ける

---

### 中盤（第6〜13ターン）— 育成と拡張

**長老評議会の圧力を抑えながら仲間思い型を育てる**フェーズです。

#### 長老評議会の圧力のコントロール

大型討伐を2〜3ターンに1本完遂すれば圧力はリセットされます。鬼殺し型を大型討伐専任として動かし続けるイメージです。

#### 英雄候補召喚の判断

```
召喚を許可する条件：
  元一行にアクティブな仲間が4名以上残っている
  財宝に100両の余裕がある
  元一行が進行中の討伐を持っていない（戦力が落ちても失敗しない）
```

召喚すると元一行から2〜3名が移ります。元一行の戦力スコアが必要値を割ることがないか必ず確認してください。

#### 鬼殺し型の一行管理

鬼殺し型の配下の士気報告値が「低下」になったら警戒サインです。実際の士気はさらに低い可能性があります（識別力が低い鬼殺し型の隊長は仲間の状態を正確に報告できない）。

対処法：

1. 仲間思い型の一行からスキルのある仲間を移す（新鮮な士気で補充）
2. 鬼殺し型を討伐から一時外して士気回復を待つ（アイドルコストは発生するが連鎖離脱よりマシ）

#### 仲間思い型配下のぬるま湯化対策

仲間思い型の配下に8ターン以上いる仲間はスキル成長が鈍化します。**定期的に鬼殺し型の一行への短期移動**を挟むことで緩みをリセットできます。1〜2ターンだけ移してまた戻す、という使い方が有効です。

---

### 終盤（第14ターン〜）— 収穫

育てた仲間思い型の隊長を複数ラインで並走させる時期です。

```
理想的な終盤の構成：
  鬼殺し型 1人 → 大型討伐専任 or 失敗討伐の立て直し役として温存
  仲間思い型 3〜4人 → 中型討伐を安定並走
```

**他村からの助っ人招集はしない**のが鉄則です。残りターンで維持費を回収できません。

加勢イベントが来たら、仲間思い型の大きい一行に加えてスキルを育てる投資として扱います。終盤の加勢は次世代への投資なので即効性は期待しないでください。

---

## 英雄の活用

スキル85以上の仲間は⚡マークで表示されます。

英雄は：

- 一行の戦力スコアを+20%引き上げる
- 他の仲間のスキル成長を+2/ターン底上げする
- 腕前が急上昇するため、放置すると高確率で去る

**英雄優遇（500両/ターン）を使うかどうかの判断基準：**

```
優遇すべきケース：
  スキルが85〜90台で、まだ伸びしろがある
  大型討伐に必要な戦力スコアの要になっている
  忠誠心がすでに20以下で離脱リスクが高い

優遇しなくてよいケース：
  士気が十分高く（65以上）離脱リスクが低い
  財宝に余裕がない序盤
```

## 覚醒の狙い方

覚醒の条件：

```
鬼殺し型の配下に3ターン以上いる
士気が30以下
胆力が70以上
スキルが40以上
未覚醒・非英雄候補
```

胆力は隠しパラメータです。仲間一覧モーダルで観測値（誤差あり）を確認できます。

**覚醒を狙う戦略：**

1. 仲間思い型の一行で胆力が高そうな仲間を見つける（観測値70以上が目安）
2. 鬼殺し型の一行に短期移動させる
3. 士気が30以下になっても離脱しなければ覚醒の可能性がある
4. 覚醒するとスキル+20〜35の急成長、英雄候補確定

ただし**失敗リスクがある**ことを忘れずに。胆力の観測値には誤差があり、実際は低い場合は普通に去ります。財宝に余裕がある中盤以降に試みるのが安全です。

## 失敗討伐対応フロー

### 失敗ペナルティの計算

放置するほどペナルティが加速します。

```
毎ターンのペナルティ = 討伐収益 × 0.05 × 失敗継続ターン数

中型討伐（5,000両）の場合：
  1ターン放置 →  250両
  2ターン放置 →  500両（累計  750両）
  3ターン放置 →  750両（累計1,500両 ≒ 討伐収益の30%）
  4ターン放置 → 1,000両（累計2,500両 ≒ 討伐収益の50%）

大型討伐（10,000両）の場合：
  3ターン放置 → 累計3,000両
  4ターン放置 → 累計5,000両（討伐収益の半分が消える）
```

**1ターンでも早く立て直すのが正解です。** 2ターン目以降はペナルティが急増します。立て直しに必要な鬼殺し型と戦力スコアが揃っていないなら、まず仲間思い型の一行から移してスコアを上げることを最優先してください。

```
討伐が失敗した
　↓
鬼殺し型の隊長を確認する
　↓
立て直しに必要なスコアを確認する
  小型 → 50以上
  中型 → 150以上
  大型 → 300以上
　↓
スコアが足りない → 仲間思い型の一行から鬼殺し型の一行へ移してスコアを上げる
　↓
鬼殺し型を立て直し担当にアサイン
　↓
次のターンに立て直し成功 → 収益の15%を回収
```

**放置は厳禁**です。失敗ペナルティは経過ターンの二乗に近い形で増加します。

## 隠しパラメータの扱い方

このゲームには3つの隠しパラメータがあります。

### 相性（chemistry）

隊長と仲間の相性。高いと士気が毎ターン+1・スキル成長+2・離脱確率×0.8。低いとその逆。**一行替えするとリセット**されるため、良い組み合わせが見つかったら安易に動かさない方がいい。

### 胆力（resilience）

鬼殺し型の配下での士気低下を緩和する能力。覚醒の鍵になるパラメータ。隊長の識別力が高いほど観測値の誤差が小さくなります。

### 識別力（discernment）

隊長が仲間のスキルと胆力をどれだけ正確に把握できるか。鬼殺し型は低め（20〜60）、仲間思い型は高め（40〜90）。識別力が低い隊長の報告値は大きくズレており、「できると思っていた仲間が実はダメだった」という事態が起きやすい。

**結果画面で全パラメータの実値が公開されます**。ゲーム中に感じた「あの離脱は予想外だった」「あの討伐で失敗したのはなぜか」の答えがそこにあります。

---

## 隊長の識別力の影響

```
識別力90以上 → 観測誤差±3（ほぼ正確）
識別力70〜89 → 観測誤差±8
識別力50〜69 → 観測誤差±15
識別力30〜49 → 観測誤差±25
識別力30未満 → 観測誤差±40（まったく当てにならない）
```

識別力の低い鬼殺し型の隊長の一行では、士気の報告値と実態がかけ離れていることがあります。「報告値は普通だったのに突然去った」のは識別力の問題である可能性が高い。

## よくある失敗パターンと対処法

**「鬼殺し型の一行が気がついたら全員去っていた」**
鬼殺し型の士気報告値は識別力が低いため、実態より良く見えることがあります。「低下」の報告が出る前に一行替えで補充を始めてください。

**「失敗中に隊長交代したらさらに悪化した」**
隊長交代は失敗リスク×1.25のペナルティがあります。交代するなら失敗前か、十分なスコアの鬼殺し型が確保できるときだけにしてください。

**「英雄候補を召喚させたら元の討伐が失敗した」**
召喚で元一行から2〜3名が移ります。討伐進行中に戦力スコアが必要値を割ると毎ターン失敗リスクが上昇します。召喚は討伐完了後のタイミングが安全です。

**「加勢を入れたら戦力が落ちて討伐が失敗した」**
加勢はスキル10〜25と非常に低い。大量に加えた直後は平均スキルが急落します。加勢後は1〜2ターン、討伐規模を落とすか、討伐から外して育成期間を設けてください。

**「仲間思い型だけで回していたら長老評議会の圧力が積み重なって破産した」**
第6ターンから大型討伐の実績がないと500両/2ターンのペナルティが永続します。鬼殺し型を温存しすぎずに定期的に大型討伐に投入してください。

**「仲間思い型配下の仲間のスキルが伸びなくなった」**
仲間思い型の配下に8ターン以上いるとスキル天井が発生します。定期的に鬼殺し型の一行への短期移動（1〜2ターン）で緩みをリセットしてください。

## 100ターンモード特有の戦略

100ターンでは世代交代が起きます。

```
第1〜20ターン  : 生存・基盤構築（20ターンモードと同じ）
第21〜50ターン : 第2世代の育成。英雄候補の子が英雄候補になる
第51〜80ターン : 一行文化の定着期。鬼殺し型多め→硬派な文化、仲間思い型多め→緩やかな文化
第81〜100ターン: 文化の収穫期。第3世代の隊長が一行を支える
```

100ターンで重要になるのは仲間思い型配下のぬるま湯化です。長期で仲間思い型だけ回すと徐々にスキル天井が全体に広がり、大型討伐が回らなくなります。**鬼殺し型を「一行の刺激剤」として定期投入**するローテーションが鍵です。

## 30秒の使い方

1ターン30秒は**ほぼ確実に足りません**。特に中盤以降、隊長が増えて討伐も複数になると、全部を確認して判断するのは不可能です。重要な情報を素早く読み取るコツを覚えてください。

---

### 最初の5秒：バナーとKPIを確認する

画面上部のバナー警告が最初の判断材料です。

```
⚠ 士気危機            → 鬼殺し型の一行に誰かを移す
⚠ 財宝が危険水域      → 討伐を慎重に選ぶ、アイドルコストは絶対NG
😌 仲間思い型配下○ターン継続 → ぬるま湯化が始まっている仲間がいる
```

KPIの数字は毎ターン必ず見てください。

```
財宝（両）    → 減り方が急なら失敗討伐か離脱コストが出ている
一行の総人数  → 前ターンより減っていたら離脱が出た
士気（報告値）→ 「低下」「危機的」なら次ターン離脱が来る
```

---

### 次の10秒：討伐カードのアイコンを読む

討伐カードは文字を読まなくてもアイコンで状況が把握できます。

```
[鬼殺し型指名] バッジ（赤）→ 仲間思い型はアサインできない。鬼殺し型が空いているか確認
戦力スコアの色
  緑  → 充足。そのままアサインしてOK
  黄  → 軽微な不足。仲間思い型が警告を出すかもしれないが進められる
  赤  → 大幅不足。失敗リスクが急上昇する。避けるか一行を補強してから
🔥 失敗中             → 最優先。鬼殺し型を投入する
進捗バー              → 残りターン数を確認。終盤の討伐は次ターンで完了するかどうかが判断材料
```

---

### 次の10秒：隊長カードの顔文字を読む

仲間の顔文字列は最速で一行の状態を把握できるUIです。

```
😄😄😄😄😄  → 健全。この討伐を続けてOK
😐😐😟😐😐  → 1〜2名が疲れ始めている。次のターンも悪化するようなら補充を検討
😩😩💀😩😟  → 危険。このターン離脱が出る可能性が高い。即補充か討伐から外す
```

⚡マークのついた仲間（英雄）がいるかも確認してください。英雄がいれば戦力スコアに+20%のボーナスがあります。

隊長名にホバーすると、その隊長の一言コメントが出ます。

```
「結果が全てだ。余計なことを考えるな。」  → 鬼殺し型。一行は消耗しているが安定稼働中
「最近、なんか刺激がないんですよね。」    → ぬるま湯化が始まっているサイン
「申し訳ない…もっと守れたはずなのに。」  → 士気が危険域の仲間思い型の一行
```

---

### 残り5秒：1アクションだけ実行する

30秒で全ての最適化をしようとしないことが大事です。**そのターンで最もリスクの高い1点だけを対処する**という優先順位を持ってください。

```
優先順位（高い順）：
  1. 失敗中討伐への鬼殺し型の投入
  2. 💀が出ている一行への補充
  3. アイドルになりそうな隊長のアサイン
  4. 戦力スコアが赤の討伐の調整
  5. 英雄候補召喚の検討
```

時間切れで自動進行した場合、アサインされていない討伐はそのまま残ります。アイドルコストだけは必ず防いでください。

---

### ポップアップへの対応方針

ポップアップは**無視せず必ず読む**のが基本ですが、内容によって判断を変えてください。

```
💥 覚醒イベント          → 必ず確認。英雄候補が確定したので召喚候補に
⭐ 英雄候補浮上          → 「後で決める」を押してもよい（4ターンのスヌーズ）
                          ただし、一行に4名以上いるなら今すぐ召喚させた方がいい
📊 長老評議会の圧力      → 必ず確認。次のターンで鬼殺し型を大型討伐に入れる計画を立てる
🎓 加勢イベント          → 30秒タイマーが別途走る。焦らず一行を選ぶ
                          財宝に余裕がなければスキップしても問題ない
😌 ぬるま湯化警告        → バナー確認だけでOK。即対処しなくても2〜3ターン以内に対応すれば間に合う
```

---

### 「戦力不足」ポップアップが出たとき

仲間思い型が警告を出した場合と、鬼殺し型が警告を出した場合で意味が異なります。

```
仲間思い型からの警告（不足率5%以上）：
  「正直、今の一行では厳しいです。」
  → 慎重に判断する価値あり。一行を補強してからアサインする選択肢を検討

鬼殺し型からの警告（不足率10%以上）：
  「やってやる。多少キツくても一行が頑張るしかない。」
  → よほどの不足でなければ強行してもよい。ただし士気-5のペナルティあり
```

「それでもアサイン」を押すと士気が-5されます。序盤の士気が高い時期はリスクが小さいですが、すでに士気が低下している一行では離脱連鎖のトリガーになることがあります。

## 4ターンごとの仲間の声を読む

4ターンに1回、条件に応じた仲間からポップアップでメッセージが届きます。これは単なる演出ではなく、**今の一行の状態を示すシグナル**です。行動に変えてください。

---

### 離脱多発サイン

```
「最近、また人が去りましたよね。正直、不安です。」
「なんか、一行が静かになってきた気がします...」
「次は自分かなって、少し考えてしまいます。」
```

**意味：** 直近2ターンで同じ一行から2名以上去った状態。連鎖離脱が始まっています。

**対応：** 次のターンで同じ一行の士気を確認。「低下」以下なら即補充か、その一行の討伐規模を落とす。

---

### 一行が健全なサイン

```
「この隊長と一緒なら頑張れます。」
「なんか、この仕事向いてるかもって思えてきました。」
「ここの一行は雰囲気がいいですよね。」
```

**意味：** 相性（chemistry）が70以上かつ士気60以上の仲間がいる。今の隊長との組み合わせが機能しています。

**対応：** この仲間は一行替えさせないこと。良い相性の組み合わせは貴重なリソースです。

### 成長実感サイン

```
「腕の立つ一行で戦えて嬉しいです。」
「ここにいると自分も伸びてる気がします。」
「最近、難しい討伐でも楽しいって思えるようになりました。」
```

**意味：** スキル80以上の仲間が出始めています。英雄候補が育っています。

**対応：** この仲間の優遇を検討するタイミング。腕前が上がりきる前に手を打つ。

### 限界サイン

```
「...正直、限界が近いかもしれないです。」
「毎朝、鬼退治に出るのが怖いです。」
「眠れない夜が増えました。」
```

**意味：** 士気30以下の仲間がいます。離脱確率は30〜55%/ターンです。

**対応：** 即座に仲間思い型の一行への移動か、鬼殺し型をそのターン討伐から外して士気回復を図る。このメッセージが来てから2ターン以内に対処しないと確実に去ります。

---

### ぬるま湯化サイン

```
「最近、なんか刺激がないんですよね。」
「このままでいいのかな、って思うことがあります。」
「居心地はいいんですけど、成長してるのかって。」
```

**意味：** 仲間思い型の配下に12ターン以上いる仲間がいます。スキル天井78が発生しています。

**対応：** この仲間を鬼殺し型の一行に短期移動（1〜2ターン）させてぬるま湯化をリセット。鬼殺し型が忙しければ、討伐完了後に移動を計画する。

---

### 鬼殺し型配下でのブレイクスルーサイン

```
「きつい。でも、なんか燃えてきた。」
「ここで折れたら終わりだと思って、踏ん張ってます。」
「悔しいけど、確実に強くなってる気がします。」
```

**意味：** 鬼殺し型の配下に3ターン以上・胆力70以上の仲間がいます。覚醒の直前の可能性があります。

**対応：** この仲間を鬼殺し型から外さないこと。あと1〜2ターンで覚醒する可能性があります。ただし離脱リスクも同時に高まっているため、腕前を確認しておく。

### 英雄流出サイン

```
「そろそろ、もっと大きいことやってみたいんですよね。」
「正直、他の村からも声がかかってて...」
「自分の腕を、もっと大きな場所で試してみたい時期です。」
```

**意味：** スキル85以上の英雄が優遇されていない状態です。次のターンに去る可能性があります。

**対応：** 即座に英雄優遇（500両/ターン）を発動。財宝に余裕がない場合でも、英雄を失うコストの方が高い。

### 覚醒後の声

```
「あの時期があったから、今の自分があると思ってます。」
「不思議ですよね。あそこまで追い込まれたのに、今は感謝してます。」
```

**意味：** 覚醒済みの仲間がいます。この仲間は一行の宝です。

**対応：** 召喚させて英雄候補→隊長へ。覚醒者が隊長になると鬼殺し型と仲間思い型のハイブリッド的な動き方をします。
