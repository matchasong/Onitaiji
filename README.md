---

あなたは村の長老。**鬼殺し型（高実績・高負荷）** と **仲間思い型（穏健・育成型）** の隊長を討伐依頼に配置し、村を守り続けてください。

## 隊長

**鬼殺し型（TYPE-A）** 😤😈😬
大型討伐（〜13,000両/ターン）を回せる唯一の存在。一行の士気を毎ターン削る。失敗討伐の立て直し役。識別力は低め。

**仲間思い型（TYPE-B）** 🤔🧐😊
中型討伐（〜7,000両/ターン）が上限。仲間を育て、英雄候補（隊長候補）を輩出する。識別力は高め。

---

## 主要な仕組み

**戦力スコア** = 人数 × 平均スキル × 士気補正
討伐依頼には必要スコアがあり、不足すると失敗リスクが毎ターン上昇します。

**離脱のロジック**
士気が低く、腕が立つ仲間ほど一行を去りやすい。優秀な者ほど先に去ります。

**長老評議会の圧力**
第6ターン以降、大型討伐の実績がないと2ターンごとに500両のペナルティ。鬼殺し型を完全に干すことはできません。

**ぬるま湯化**
仲間思い型の配下に長くいる仲間はスキル成長が鈍化します（8ターン→天井83、12ターン→78、16ターン→72）。定期的に鬼殺し型への一行替えが必要です。

**覚醒**
鬼殺し型の配下で3ターン以上耐え抜き、胆力が高い仲間が稀に覚醒。スキルが急成長し英雄候補として浮上します。
